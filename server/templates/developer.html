Use the Confer APIs for querying user's preferences, recommendations and network. Leverage confer data for building conference tools.

Public Data

For any conference you can get all the public data here -- http://confer.csail.mit.edu/static/conf/<conf-name>/data/ For eg:

  • CHI 2014 Data: http://confer.csail.mit.edu/static/conf/cscw2014/data/
  • CHI 2013 Data: http://confer.csail.mit.edu/static/conf/chi2014/data/

User's Preference Data

You can also get a user's personal data through confer-connect if the user authorize your app to collect his/her data.

  • Step 1: Create a confer app at http://confer.csail.mit.edu/developer/apps
  • Step 2: Call confer APIs with your app token to get user's private data:
    • Python Example: https://github.com/abhardwaj/confer/tree/master/examples
    • PHP Example: https://github.com/abhardwaj/confer/tree/master/examples
